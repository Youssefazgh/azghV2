# 🏫 Geräteverwaltung Hochschule Bochum

Eine moderne Webanwendung zur Verwaltung von 900 Geräten der Hochschule Bochum.

![Hochschule Bochum](https://via.placeholder.com/800x200/dc3545/ffffff?text=Hochschule+Bochum+Ger%C3%A4teverwaltung)

## 🌐 Live Demo

**Website:** [https://youssefazgh.github.io/azgh/](https://youssefazgh.github.io/azgh/)

## 📋 Features

### 🔧 Geräteverwaltung
- **900 Geräte**: Device 1 bis Device 900
- **Echtzeit-Status**: Verfügbar ⇄ Ausgeliehen mit einem Klick
- **Detail-Seiten**: Vollständige Informationen zu jedem Gerät
- **Persistente Daten**: Status wird automatisch gespeichert

### 🔍 Suche & Navigation
- **Suchfunktion**: Schnelle Suche nach Geräte-Nummer oder Raum
- **Deep-Linking**: Direkte URLs zu jedem Gerät (`#device42`)
